Prime Factorization

問題文

整数Nが与えられます。Nを素因数分解してください。

入力

入力ファイルは次の形式です。
T
N1
N2
...
NT
Tは200以下、2以上Nは1000以下です。

入力ファイル

出力

T行出力してください。i行目にはNiを素因数分解した結果をNi = p1 * p2 * ... * pk (p1 <= p2 <= ... <= pk)とするとき、
p1 p2 ... pk
の形式で出力したファイルを作成してください。

補足

各行では、末尾の空白は許容されません。注意してください。
